Key idea: Each row of the input matrix is one sample — a flattened 28×28 image, so 784 columns (pixels). The network transforms each row independently through its layers. The output matrix has the same number of rows (one per sample) but only 10 columns — one score per class. The shape goes from (batch, 784) to (batch, 10).
